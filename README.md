  # ITB Berlin 2026 Landing Page

This repository contains the static landing page for the easyGDS campaign at ITB Berlin 2026.

## Overview

The landing page is designed to promote the "100 Free Booking Engines" offer for travel agents attending ITB Berlin. It features:

*   **Hero Section**: Compelling value proposition and call to action.
*   **Features**: details on the booking engine, website, and content.
*   **Registration Form**: Embedded form for agents to claim their spot.
*   **FAQ**: Common questions and answers.

## Deployment

This site is deployed via Cloudflare Pages.

*   **Live URL**: [https://itb.easygds.com](https://itb.easygds.com)
